# Code Of Conduct

This project is for practical, respectful collaboration.

## Expected Behavior

- Be kind and direct.
- Assume good intent when clarifying mistakes.
- Keep feedback about the work, not the person.
- Welcome first-time contributors and small documentation fixes.
- Respect maintainers' time and project scope.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or exclusionary behavior.
- Publishing private contact information without permission.
- Repeatedly pushing off-topic or out-of-scope requests after maintainers have
  clarified the boundary.

## Enforcement

Maintainers may edit, hide, close, lock, or delete issues, discussions, pull
requests, or comments that violate this code of conduct. Maintainers may also
block users from participating in the project when needed to protect the
community.

## Reporting

If you need to report a conduct issue, open a minimal issue asking for a private
contact path. Do not include sensitive details in a public issue.
